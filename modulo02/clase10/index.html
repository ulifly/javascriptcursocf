En JavaScript, los operadores lógicos son utilizados para realizar operaciones lógicas en expresiones booleanas. Los operadores lógicos en JavaScript son los siguientes:
  • AND lógico (&&): Devuelve true si ambos operandos son true. Por ejemplo: true && false devuelve false.
  • OR lógico (||): Devuelve true si al menos uno de los operandos es true. Por ejemplo: true || false devuelve true.
  • NOT lógico (!): Invierte el valor de un operando booleano. Si el operando es true, devuelve false, y si el operando es false, devuelve true. Por ejemplo: !true devuelve false.
  • Operador nullish coalescing: asigna un valor por defecto a una variable en caso de que el valor original sea null o undefined null ?? valorPorDefecto; devuelve valor por defecto
Estos operadores lógicos son útiles para combinar y evaluar condiciones lógicas en expresiones condicionales, bucles y otras estructuras de control en JavaScript.
