Ciclos en JavaScript
En JavaScript, los ciclos se utilizan para ejecutar un bloque de código repetidamente hasta que se cumpla una condición específica.

Hay varios tipos de ciclos en JavaScript, como el ciclo for, el ciclo while y el ciclo do...while.

ciclo for

El ciclo for se utiliza cuando se conoce de antemano la cantidad exacta de veces que se desea ejecutar un bloque de código. La sintaxis básica del ciclo for es la siguiente:
for ( let i = 0; i < 11; i ++){
console.log(i);
}
console.log( 'Fin' );

En la inicialización, se declara e inicializa una variable de control.

La condición se evalúa antes de cada iteración y, si es verdadera, se ejecuta el bloque de código. Después de cada iteración, se ejecuta la expresión de incremento. El ciclo for se repite hasta que la condición se evalúe como falsa.

Ciclo while:

El ciclo while se utiliza cuando no se conoce de antemano la cantidad exacta de veces que se desea ejecutar un bloque de código.
while(cont != 0) {
console.log(cont);
cont = cont -1
}
La condición se evalúa antes de cada iteración y, si es verdadera, se ejecuta el bloque de código. El ciclo while se repite hasta que la condición se evalúe como falsa.

Ciclo do...while:

El ciclo do...while es similar al ciclo while, pero la diferencia es que el bloque de código se ejecuta al menos una vez antes de evaluar la condición.
do {
console.log(iteracion);
iteracion = iteracion + 10;
} while (iteracion < 100 );
Este bucle do...while ejecutará el código dentro de las llaves al menos una vez y luego se repetirá siempre que la condición iteracion <= 100 sea verdadera. En cada iteración, el valor de iteracion se incrementa en 10 y se muestra en la consola. El bucle se detendrá cuando iteracion sea mayor que 100 .
