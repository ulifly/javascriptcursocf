En JavaScript, las condiciones se utilizan para tomar decisiones y ejecutar diferentes bloques de código según el resultado de una expresión booleana. Las condiciones se pueden implementar utilizando la declaración if, else if y else, así como el operador condicional ?.
La declaración if se utiliza para ejecutar un bloque de código si una condición especificada es verdadera. Por ejemplo:
if (edad >= 18) {
console.log("Eres mayor de edad");
}
En el ejemplo anterior, si la variable edad es mayor o igual a 18, se imprimirá en la consola el mensaje "Eres mayor de edad".
La declaración else if se utiliza para especificar una nueva condición que se evaluará si la condición anterior es falsa. Por ejemplo:
if (edad >= 18) {
console.log("Eres mayor de edad");
else if (edad >= 13) {
}
console.log("Eres adolescente");
} else {
console.log("Eres niño");
}

En este caso, si la variable edad es mayor o igual a 18, se imprimirá "Eres mayor de edad".

Si no se cumple esa condición pero la edad es mayor o igual a 13, se imprimirá "Eres adolescente". Si ninguna de las condiciones anteriores es verdadera, se imprimirá "Eres niño".

El operador condicional ? es una forma abreviada de escribir una declaración if...else. Por ejemplo:
let resultado = (edad >= 18) ? "Eres mayor de edad" : "Eres menor de edad";
console.log(resultado);
